#  Cloud-Based OneNote(Cloud-Based Note Taking App)

A modern cloud-based note-taking web application built using "React.js" and "Firebase" that allows users to securely create, manage, search, and organize notes online from anywhere.

---

##  Live Demo

🌐 https://notes-app-73c7d.web.app

---

##  Project Overview

Cloud-Based OneNote is a responsive and user-friendly web application designed for efficient note management.  
The application provides secure authentication, cloud storage, real-time database integration, and an elegant UI for better user experience.

This project was developed to demonstrate:
- Frontend development using React.js
- Firebase Authentication
- Firestore cloud database integration
- CRUD operations
- Responsive UI/UX design
- Real-time cloud-based note management

---

##  Features

-  User Authentication (Login & Signup)
-  Google Sign-In Authentication
-  Create Notes
-  Edit Notes
-  Delete Notes
-  Search Notes
-  Cloud Storage using Firebase Firestore

-  Secure Logout Functionality

##  Installation & Setup

### 1️. Clone the Repository

git clone https://github.com/anushreeurao/Cloud-Based-Note-Taking-App

---

### 2️. Navigate to Project Directory

cd Cloud-Based-Note-Taking-App

---

### 3️. Install Dependencies

npm install

---

### 4️. Start the Development Server

npm start

---
